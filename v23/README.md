# 🦅 Projeto Águia — Versão 23 (Release Estável)

## Sobre

Primeira versão consolidada do Projeto Águia, unificando:

- Catecismo Anual no padrão aprovado da V19;
- Academia com visual restaurado da V18;
- Minha Jornada como tela inicial;
- OCR de boletim;
- IA híbrida;
- Painel do Responsável sincronizado.

## Política de estabilidade

A partir desta versão, o módulo **Catecismo Anual** passa a ser considerado estável, recebendo apenas correções pontuais.
