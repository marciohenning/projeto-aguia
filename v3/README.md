# Projeto Águia - Versão 3

## Data

Junho de 2026

## Objetivo

Transformar o Projeto Águia em uma plataforma completa de acompanhamento escolar, disciplina e desenvolvimento pessoal para um aluno do 9º ano do Ensino Fundamental.

## Principais funcionalidades

* Tela inicial com senha familiar.
* Dashboard inteligente.
* Controle diário de tarefas.
* Visão semanal das atividades.
* Calendário mensal interativo.
* Planejamento até dezembro.
* Boletim inteligente baseado nas notas.
* Priorização automática das disciplinas com menor desempenho.
* Painel do responsável.
* Sistema de recompensas semanais.
* Sistema de medalhas e conquistas.
* Acompanhamento das orações em latim.
* Integração do estudo diário de inglês com Rosetta Stone.
* Exportação do progresso em arquivo JSON.

## Melhorias em relação à Versão 2

* Adicionado login por senha.
* Adicionado painel do responsável.
* Adicionado contador da missão até dezembro.
* Adicionado boletim inteligente.
* Adicionado controle de faltas.
* Adicionado sistema de backup dos dados.
* Melhor organização das telas.
