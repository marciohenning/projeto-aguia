# Objetivo

O Projeto Águia é uma plataforma de formação integral desenvolvida para acompanhar o crescimento acadêmico, humano, financeiro e espiritual de João Miguel durante o 9º ano do Ensino Fundamental.

Esta versão organiza todo o aprendizado segundo a tradição clássica do **Trivium e Quadrivium**, conciliando:

- Boletim escolar real;
- Currículo em Movimento do GDF;
- Inglês (Rosetta Stone);
- Latim;
- Educação Financeira;
- Catecismo Menor de São Pio X;
- Sistema diário de recompensas.

---

# Currículo GDF

O Currículo em Movimento do Distrito Federal é utilizado como referência para antecipar os conteúdos mais prováveis do 3º e 4º bimestres.

Entre eles:

### Matemática

- Porcentagem
- Equações
- Funções
- Estatística
- Geometria

### Português

- Interpretação textual
- Produção textual
- Gramática contextualizada

### Ciências

- Genética
- Ecologia
- Corpo humano

### História

- República Brasileira
- Guerras Mundiais

### Geografia

- Globalização
- Geopolítica

### Inglês

- Reading
- Vocabulário
- Verbos
- Interpretação

# Catecismo Menor de São Pio X

Todos os sábados e domingos são dedicados à formação cristã.

Sequência de estudo:

1. Doutrina Cristã
2. Credo
3. Oração
4. Mandamentos
5. Sacramentos
6. Virtudes Cristãs

Inclui:

- leitura;
- perguntas e respostas;
- orações;
- latim;
- reflexão semanal.
