# 🦅 Projeto Águia — Versão 28

## Novidade principal

A Versão 28 acrescenta ajustes de responsividade para melhorar o uso em celulares Android e iOS, nos navegadores:

- Google Chrome;
- Safari;
- Firefox;
- navegadores móveis compatíveis.

## Melhorias de responsividade

- Ajuste do `viewport` com suporte a áreas seguras do iPhone.
- Melhor adaptação para telas pequenas.
- Cartões ocupando 100% da largura disponível.
- Tabelas com rolagem horizontal.
- Calendário anual mais confortável no celular.
- Botões e campos com fonte adequada para evitar zoom automático no iOS.
- Barra inferior ajustada para celulares.
- Suporte a `safe-area-inset` em iPhones.
- Imagens, iframes e prévias de boletim adaptadas à tela.

## Mantido das versões anteriores

- Academia com troca livre de disciplina.
- Prioridade inteligente.
- Botão **🦅 Escolher para mim**.
- Conquistas no padrão visual da V18.
- Catecismo Anual preservado.
- Calendário com referências pedagógicas.
- Boletim OCR.
- Painel do Responsável.
