# Projeto Águia V35.2.4 Estável

## Objetivo

O Projeto Águia é uma plataforma de reforço escolar, formação cristã e educação financeira desenvolvida para acompanhar o progresso do estudante de forma organizada e motivadora.

Lema:

> Cognoscere, amare et servire Deum.
> (Conhecer, amar e servir a Deus)

---

## Principais Recursos

### Minha Jornada
- Resumo geral do progresso.
- Acompanhamento diário.
- Visualização rápida das atividades.

### Missão
- Tarefas diárias.
- Aprovação pelo responsável.
- Sistema de recompensas.

### Academia
- Exercícios por disciplina.
- Correção automática.
- Estatísticas de desempenho.
- Quantidade configurável de questões:
  - 5
  - 10
  - 20
  - 50
  - 100
  - 200
  - 500

### Disciplinas Disponíveis

- Matemática
- Português
- Ciências
- História
- Geografia
- Inglês
- Educação Financeira
- Catecismo
- Latim
- Geometria

### Catecismo
- Catecismo Menor de São Pio X.
- Progresso anual.
- Leitura organizada por etapas.

### Calendário
- Planejamento mensal.
- Planejamento anual.
- Organização dos estudos.

### Conquistas
- Sistema de evolução.
- Registro de metas alcançadas.

### Boletim
- Registro das notas escolares.
- Acompanhamento por bimestre.
- Importação de documentos.

### Responsável
- Acompanhamento do progresso.
- Aprovação das missões.

### Backup
- Exportação dos dados.
- Restauração dos dados.
