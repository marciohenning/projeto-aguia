# 🦅 Projeto Águia — Versão 34

## Correção principal

A Versão 34 normaliza a relação entre:

- Missão do Dia;
- aprovação do responsável;
- XP;
- Conquistas;
- Mapa da Jornada.

Agora, quando o responsável aprova a missão, o sistema atualiza corretamente o progresso das conquistas e o nível atual da jornada.

## Melhorias

- Missões aprovadas contam para conquistas.
- XP da missão atualiza o mapa da jornada.
- Progresso geral das conquistas mostra:
  - missões aprovadas;
  - tarefas concluídas;
  - catecismo concluído;
  - XP total.
- Mapa da Jornada mostra nível atual e próximo objetivo.
- Barras de progresso normalizadas.

## Mantido da Versão 33

- Navegação PWA simplificada.
- Tela inicial com atalhos.
- Academia com escolha livre.
- Catecismo Anual preservado.
- Boletim OCR.
- Painel do Responsável.
