# 🦅 Projeto Águia - Versão 18

## Novidades

- O aplicativo sempre inicia pela aba **Minha Jornada**.
- O Catecismo Anual foi reorganizado de forma mais pedagógica.
- As lições são divididas por blocos completos de perguntas e respostas.
- As numerações, perguntas e subtítulos aparecem em destaque.
- O primeiro sábado do ano corresponde à **Lição 1**.
- Cada sábado seguinte avança uma lição até completar 52.
- Mantém Conquistas, IA, Calendário Anual e Painel do Responsável.
