# Projeto Águia - Versão 4

## Objetivo

Transformar o Projeto Águia em uma plataforma de aprendizagem gamificada, não apenas uma lista de tarefas.

## Novidades da V4

- Exercícios por matéria com certo e errado.
- Banco inicial de questões para Inglês, Latim, Português, Matemática, Ciências, História, Geografia, Arte e Projeto Interdisciplinar.
- Desafio diário com bônus de XP.
- Sistema de XP.
- Sistema de níveis:
  - Nível 1 — Aspirante
  - Nível 2 — Escudeiro
  - Nível 3 — Guardião
  - Nível 4 — Cavaleiro
  - Nível 5 — Águia de Ouro
- Sistema de vidas.
- Sequência de acertos.
- Relatório semanal para o responsável.
- Desempenho por matéria.
- Recomendação automática da matéria a reforçar.
- Exportação de progresso.
