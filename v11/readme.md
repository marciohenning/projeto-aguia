# 📖 Sobre o Projeto

O Projeto Águia é uma plataforma de formação integral desenvolvida para acompanhar o crescimento acadêmico, intelectual, moral e espiritual de João Miguel durante o 9º Ano do Ensino Fundamental.

O aplicativo integra:

* Currículo em Movimento do GDF;
* Planejamento baseado no boletim escolar;
* Trivium e Quadrivium;
* Inglês (Rosetta Stone);
* Latim;
* Educação Financeira;
* Catecismo Menor de São Pio X;
* Sistema de recompensas;
* Inteligência Artificial de apoio ao estudo.

---

# 🚀 Principal novidade da V11

## ✝️ Catecismo Anual de São Pio X

A Versão 11 transforma o estudo do Catecismo em uma jornada anual.

Todos os sábados possuem uma leitura programada, distribuída automaticamente ao longo dos 52 sábados do ano.

Cada leitura possui:

* tema da semana;
* páginas programadas;
* leitor integrado;
* progresso salvo automaticamente;
* XP de Catecismo;
* medalhas;
* integração com o Tutor Águia IA.

# 🤖 Planejador Águia IA

A Inteligência Artificial do aplicativo acompanha:

* notas escolares;
* tarefas concluídas;
* XP acumulado;
* progresso no Catecismo;
* calendário anual;
* disciplinas prioritárias.

Com essas informações, o sistema gera automaticamente sugestões para o próximo dia de estudo.

Também é possível copiar um relatório para utilização em assistentes de IA externos.
