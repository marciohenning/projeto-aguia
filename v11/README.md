# 🦅 Projeto Águia - Versão 11

## Novidade principal

A V11 adiciona a aba **Catecismo Anual de São Pio X**.

A leitura é distribuída entre os 52 sábados do ano, com:

- tema semanal;
- páginas sugeridas;
- leitor interno;
- progresso anual;
- XP de Catecismo;
- medalhas;
- revisão dominical;
- integração com o Planejador Águia IA.

## Observação

Esta versão inclui um leitor interno com trechos e temas do Catecismo.  
Para reproduzir o PDF completo dentro do app, coloque o arquivo PDF na pasta `v11` e adicione um iframe apontando para ele.
