# 🦅 Projeto Águia - Versão 19

## Novidades

- Academia com perguntas rotativas, evitando repetir perguntas já acessadas.
- Tutor Águia IA Local para consultas simples, sem servidor externo.
- Aba Boletim com tabela de notas do 1º, 2º, 3º e 4º bimestre.
- Importação/visualização de boletins em PDF, PNG, JPG e JPEG.
- Exportação e importação das notas em JSON.
- Dados salvos no navegador com localStorage.
- Mantém Minha Jornada, Catecismo Anual, Conquistas, Calendário e Painel do Responsável.

## Observação

A importação de PDF/imagem permite visualizar o boletim dentro do app.  
A leitura automática das notas por OCR exigiria uma biblioteca pesada ou uma IA externa/API. Nesta versão, as notas são digitadas na tabela e ficam salvas no navegador.
