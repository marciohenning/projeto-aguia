# 🦅 Projeto Águia - Versão 13

## Novidade principal

A V13 transforma o Catecismo em um leitor completo por sábado.

## Estrutura nova

A pasta `catecismo/` contém 52 arquivos:

- `semana01.html`
- `semana02.html`
- ...
- `semana52.html`

Ao clicar em **Abrir lição**, o aplicativo carrega automaticamente o arquivo correspondente àquele sábado.

## Funcionalidades

- Leitura completa de cada sábado.
- Busca por tema.
- Favoritos.
- Progresso dos 52 sábados.
- XP de Catecismo.
- Link direto entre Calendário Anual e lição do Catecismo.
- Relatório para IA externa.
- Funciona como PWA.
