## Novidades

- Correção dos botões de matérias na Academia.
- Botões agora mostram ícones e nomes das disciplinas.
- Nova aba IA com automação local.
- Planejador Águia IA gera plano automático de estudo.
- Relatório copiável para usar com IA externa.
- Nova aba Conquistas.
- Modo campanha com níveis.
- Mantém Calendário Anual, Minha Jornada, Trivium, Quadrivium e Catecismo.

## Observação sobre IA

A V10 usa IA local baseada em regras dentro do navegador.  
Ela não usa chave de API para proteger dados e evitar exposição de credenciais no GitHub Pages.

Para usar uma IA externa, copie o relatório da aba IA e cole no ChatGPT ou outro assistente.
