# 🦅 Projeto Águia - Versão 8

## Principal novidade
Calendário anual de estudos.

Agora o app permite visualizar, mês a mês, a distribuição das matérias e eixos pedagógicos:

- Gramática
- Dialética
- Retórica
- Quadrivium
- Síntese
- Catecismo

Cada dia do calendário anual pode ser clicado para exibir:

- eixo do dia;
- matérias;
- tarefas;
- missão correspondente.

## Mantém da V7

- Boletim de João Miguel como prioridade;
- Currículo do GDF como subsídio;
- Trivium e Quadrivium;
- Catecismo aos sábados e domingos;
- missão diária valendo R$ 10;
- calendário mensal;
- visão semanal;
- painel do responsável.
