# Sobre o Projeto

O Projeto Águia é uma plataforma de formação integral criada para acompanhar o desenvolvimento acadêmico, intelectual, moral e espiritual de João Miguel durante o 9º Ano do Ensino Fundamental.

A proposta une:

- Currículo em Movimento do GDF;
- Boletim escolar real;
- Trivium e Quadrivium;
- Inglês (Rosetta Stone);
- Latim;
- Educação Financeira;
- Catecismo Menor de São Pio X;
- Sistema diário de recompensas.

---

# Novidade da Versão 8

## 📆 Calendário Anual Inteligente

A V8 introduz um painel anual que permite visualizar toda a jornada de estudos ao longo do ano.

Cada dia possui uma programação previamente organizada, permitindo ao responsável e ao aluno acompanhar a evolução contínua.

Ao clicar em qualquer data, o aplicativo exibe:

- eixo pedagógico do dia;
- matérias estudadas;
- tarefas programadas;
- missão diária;
- progresso realizado.

# Currículo em Movimento (GDF)

O aplicativo utiliza o Currículo do GDF como referência para antecipar os conteúdos do 3º e 4º bimestres.

Exemplos:

## Matemática

- Porcentagem;
- Equações;
- Funções;
- Estatística;
- Geometria.

## Português

- Interpretação textual;
- Produção textual;
- Gramática contextualizada.

## Ciências

- Genética;
- Ecologia;
- Corpo Humano.

## Inglês

- Reading;
- Vocabulário;
- Verbos;
- Interpretação.
