# 🦅 Projeto Águia - Versão 6

## Lema

> **Cognoscere, amare et servire Deum.**
>
> *Conhecer, amar e servir a Deus.*

---

# Missão

O Projeto Águia é uma plataforma de formação integral criada para desenvolver:

- excelência acadêmica;
- disciplina;
- responsabilidade;
- educação financeira;
- hábitos de estudo;
- crescimento humano e espiritual.

Mais do que melhorar notas, o objetivo é construir constância e caráter.

---

# Filosofia

Todo dia é uma missão.

A recompensa não é pelas notas, mas pelo compromisso diário.

**Missão cumprida → recompensa conquistada.**

---

# Sistema de Missões

Cada dia útil possui uma lista de tarefas.

Exemplo:

- Português
- Inglês
- Rosetta Stone
- Projeto Interdisciplinar
- Latim
- Desafio Diário

Ao concluir todas as tarefas e receber aprovação do responsável:

```
🦅 MISSÃO CONCLUÍDA

⭐ +10 XP

💰 +R$ 10,00

🔥 +1 dia de sequência
```

---

# Sistema Financeiro

Cada missão aprovada libera:

## R$ 10,00

Distribuição automática:

| Destino | Valor |
|----------|---------|
| Uso pessoal | R$ 7,00 |
| Meta financeira | R$ 2,00 |
| Caridade/Formação | R$ 1,00 |

---

# Tutor Águia (IA Local)

O aplicativo analisa automaticamente:

- boletim;
- histórico de erros;
- frequência;
- sequência diária;
- desempenho nas matérias;
- revisões pendentes.

E gera recomendações como:

> Hoje a prioridade é Inglês.

> Faça 5 exercícios e 20 minutos de Rosetta Stone.
