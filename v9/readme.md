# Sobre o Projeto

O Projeto Águia é uma plataforma de formação integral desenvolvida para acompanhar a evolução acadêmica, intelectual, moral e espiritual de João Miguel durante o 9º Ano do Ensino Fundamental.

O aplicativo integra:

* Currículo em Movimento do GDF;
* Boletim escolar;
* Trivium e Quadrivium;
* Inglês (Rosetta Stone);
* Latim;
* Educação Financeira;
* Catecismo Menor de São Pio X;
* Sistema gamificado de recompensas.

---

# ✨ Novidade da Versão 9

## 🦅 Minha Jornada

A V9 introduz uma nova experiência de acolhimento para crianças e adolescentes.

O aplicativo agora possui um guia interativo que ensina, de maneira simples e divertida, como utilizar todas as funcionalidades.

A jornada apresenta:

* quem é a Águia;
* como funcionam as missões;
* como conquistar a recompensa diária;
* como utilizar o calendário anual;
* o papel do Tutor Águia;
* XP e medalhas;
* Inglês e Latim;
* o verdadeiro objetivo do projeto.

O tutorial é exibido automaticamente na primeira utilização e pode ser acessado novamente a qualquer momento.
